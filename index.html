Express.js Star Wars Movie app

After spending some time making apps on Codepen from Javascript and jQuery I've done research into frameworks like Angular, React and Express.js for building more complex web applications and node.js for the backend process.

I found a youtube tutorial on Express.js for building a web application for the Star Wars movie collection with posters, descriptions and a list of each movies main charachters.

The app runs on a express.js template which routes together all the .ejs files with an index.file. The main page has a directory to each movies page of which are all set up on a singular template page instead of 6 individual html files it generates the same template page with all corresponding information and images for the specific movie.

Movies For The Weather

My first Codepen project came from making a weather app for Free Code Camp which uses JSON data from a weather api on Darksky.net.

Using a online IP finder I can find the users latitude and longitude to determine the users location. Using the location it can then send a request to Darksky to retrieve weather information in the form of JSON data. With this data the app will then determine a style and genre of movie to watch based on the user's location weather and time of day.
